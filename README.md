# Harjoitustyö

Tämä on harjoitustyö Helsingin yliopiston kurssille "Aineopintojen harjoitustyö: Algoritmit ja tekoäly".

Työni aihe on [15-pelin](https://fi.wikipedia.org/wiki/15-peli) ratkaiseminen.
